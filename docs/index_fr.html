Politique de confidentialité – BalanceBook

Bienvenue dans l’application BalanceBook. Nous nous engageons à protéger votre vie privée et à sécuriser vos informations personnelles. Voici les détails sur la collecte, l’utilisation et le stockage des données :

1. Informations que nous collectons

  • Données personnelles : nom, adresse e-mail et numéro de téléphone.
  • Données de compte et transactions financières : montants des revenus, dépenses et dates.

2. Comment nous utilisons vos données

Nous utilisons ces données pour améliorer l'expérience utilisateur, fournir un support technique et analyser l'utilisation afin de développer de meilleures fonctionnalités financières.

3. Stockage et protection des données

Toutes les données sont stockées localement dans une base de données SQLite sur votre appareil. Nous ne transmettons pas ces données à un serveur externe sauf si vous y consentez explicitement (par exemple, en vous abonnant à une version cloud à l’avenir). Les données dans la base SQLite sont chiffrées à l’aide de mécanismes locaux.

4. Autorisations

  • INTERNET : pour télécharger les mises à jour et les sauvegardes à l'avenir.
  • Aucune autre autorisation “dangereuse” n’est demandée dans cette version.

5. Partage des données

Nous ne partageons pas vos informations personnelles avec des tiers sans votre consentement explicite. En cas de déconnexion ou de synchronisation automatique, vos données financières et personnelles peuvent être téléchargées vers des services externes tels que Firebase pour assurer leur sauvegarde et leur synchronisation entre vos appareils. Nous utilisons ces services en respectant pleinement les normes de sécurité et de confidentialité pour protéger vos données.

6. Droits des utilisateurs

  • Vous avez le droit de consulter toutes vos données dans l'application et de les supprimer (via le bouton “Effacer les données” dans les paramètres).
  • Pour demander la suppression définitive de toutes vos données, veuillez nous contacter par e-mail : balancebook001@gmail.com.

7. Mise à jour de la politique

Cette politique peut être mise à jour de temps à autre. La date de “Dernière mise à jour” sera indiquée en haut de cette page.
